## Featured Project

### [Second Brain](https://github.com/Kerr1st/Second-Brain)

Second Brain is a local-first persistent memory layer for AI agents and personal pattern discovery. It captures conversations and knowledge sources from tools like AI chat sessions, Slack-related data, Quick Desktop, YouTube, web articles, and documents, then turns them into typed, embedded memories.

Those memories are stored in PostgreSQL with pgvector and retrieved through hybrid search, combining semantic similarity with keyword search and reranking. The system exposes memory operations through MCP tools so agents can search, read, write, connect, and reuse context across sessions.

Its synthesis pipeline runs background dream cycles that review prior work for recurring patterns, contradictions, stale assumptions, open questions, and cross-project insights. The strongest findings are stored as new memories and surfaced through proactive briefing workflows.

The project includes migrations, tests, architecture docs, scheduled jobs, retrieval-quality evaluation, MCP tool contracts, and backend adapter patterns for Claude Code and Codex-style agent workflows.

Why it matters: Second Brain gives me hands-on, repo-level empathy for the systems, workflows, and governance patterns customers need when they move from AI experimentation into repeatable work.
