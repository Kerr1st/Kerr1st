## Featured Project

### [Second Brain](https://github.com/Kerr1st/Second-Brain)

A personal AI memory system for agentic workflows. It explores how AI agents can use persistent context across sessions instead of starting from a blank slate every time.

The project uses PostgreSQL, pgvector, hybrid retrieval, MCP, Kiro-style specs, Claude Code and Codex backend adapter patterns, migrations, scheduled jobs, and tests to support capture, retrieval, synthesis, and proactive briefing workflows.

Why it matters: it gives me practical, repo-level empathy for the developer workflows AI tooling companies are helping customers adopt: specs, code reviewability, tests, architecture docs, agent instructions, MCP tools, retrieval quality, and operational discipline.
